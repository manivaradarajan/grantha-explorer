## 8. Success Criteria (Phase 0)

**Functional Requirements Met:**

- All 10 Principal Upanishads + additional Visistadvaita Upanishads readable (Devanagari only)
- Three Visistadvaita bhashyas (Vedanta Desika, Rangaramanuja, Kuranarayana) displayed for all texts
- Cross-text references clickable and functional
- Bottom sheet commentary on mobile (~70% height, swipeable, navigation arrows)
- Desktop 3-column layout displays all commentaries simultaneously
- Tablet 2-column hybrid layout functional
- Column resizing with visual dividers (widths persist)
- Passage fragments in left nav
- Variant readings work (badge, selectable, differences highlighted)
- Prefatory material accessible at top of structure
- Session persistence (last verse + all preferences remembered)
- Help accessible via hamburger menu

**Performance Targets:**

- Page load <2 seconds on 4G
- Navigation <500ms
- Smooth scrolling (60fps)
- Bottom sheet animation <300ms

**UX Targets:**

- Users discover all features within 2-3 interactions
- Primary text clearly readable (no clutter)
- Keyboard navigable
- Mobile usable (no shrinking, no horizontal scroll)

**Content Quality:**

- All texts accessible and complete
- No obvious OCR errors
- Provenance metadata present (source, commit)

---

## 9. Out of Scope (Phase 0)

**NOT in Phase 0:**

- Full-text search
- Multiple English translations
- Script selector (Devanagari only)
- Language toggle (no English in Phase 0)
- User accounts / authentication
- Bookmarking, highlighting, annotations
- Analytics / tracking
- Font selection
- Audio / multimedia
- Print / PDF export
- Commentaries as selectable granthas (Phase 1+)
- Offline functionality
- Multiple commentary traditions (only Visistadvaita)

**Phase 1+ Features:**

- English translations for all texts/commentaries
- Script conversion (Roman/IAST, Kannada, and other scripts)
- Full-text search
- User accounts & sync
- Commentaries-as-granthas
- Other commentary traditions (Shankara, Madhva, etc.)
